# Count Tree Road

## Description

A counter for days gone by.

## Requirements

1. Node.js and NPM
2. Yarn
3. Elm

## Preparing

1. Run `yarn install` to install Node.js packages.
2. Run `elm-package install` to install Elm packages.

# Running

Run `yarn client` to run the web-app at [localhost:3000](http://localhost:3000).
